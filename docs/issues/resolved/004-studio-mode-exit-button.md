## 이슈 004: 스튜디오 모드 진입 시 나가기 버튼 부재

### 현상

- 작품 상세 페이지에서 '수정' 또는 '리믹스'를 클릭하여 스튜디오(스케치) 화면으로 이동했을 때, 원래 보던 갤러리로 돌아가는 명시적인 '닫기' 또는 '취소' 버튼이 없음.
- 사용자는 상단 로고나 'Gallery' 탭을 직접 클릭해야만 복귀가 가능함을 인지해야 함.

### 원인 및 분석

- 초기 설계 시 모드 전환을 간단히 처리하면서, 특정 작업(수정/리믹스)에 대한 전용 내비게이션 처리가 미흡했음.
- 모바일 환경이나 초보 사용자에게는 "어떻게 돌아가지?"라는 심리적 불안감이나 '갇힌 느낌'을 줄 수 있음. (UX 저해 요인)

### 검토 중인 해결 방안

1. **스튜디오 화면 내 '취소' 버튼 추가**:
   - 좌측 상단 또는 툴바 근처에 `← Back` 또는 `× Cancel` 버튼 배치.
   - 클릭 시 `setViewMode('feed')`를 호출하여 이전 상태로 복귀.
2. **수정 모드 전용 인디케이터**:
   - 수정 중임을 알리는 텍스트와 함께 '수정 취소' 버튼을 눈에 띄게 배치.

### 참고 사항

- 현재 갤러리 탭 클릭으로 우회가 가능하므로 우선순위는 낮음.
- 향후 스튜디오 모드 UI 고도화 시 통합 검토 권장.
